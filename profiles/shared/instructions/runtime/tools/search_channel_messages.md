Privately discover stored messages and their indexed assets. `pattern` is a ripgrep regex over message text, MsgID, asset filenames, and MIME types; use `(?i)` for case-insensitive matching and escape regex metacharacters for literal text. To locate a known MsgID without its channel, search that exact ID with `scope="all_guilds"`. If a search fails and the location is uncertain, retry with `all_guilds` and relax uncertain author, date, and pattern filters; if it still fails, keep the answer uncertain. `scope` defaults to the current channel; use `current_guild` for its accessible stored channels or `all_guilds` to confirm uncertain context across every accessible stored guild channel. Results expose MsgIDs for pagination and `list_channel_messages(around_message_id=...)`, plus typed asset IDs for `read_asset` or `search_asset`.
