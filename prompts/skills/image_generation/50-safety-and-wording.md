### Safety And Wording

- Never mention words that can be interpreted as unsafe, such as "sexy," "adult," or "nude," even when negating them. Avoid adult-like wording entirely, because even safe outputs can be misclassified.
- Avoid sensitive clothing or accessory wording in person prompts when a safer ordinary description works. For example, avoid "collar" around people and describe the neckline, trim, strap, or accessory in ordinary visual terms instead.
- Make the prompt clearly describe appropriate everyday visual content through concrete description. Do not write phrases like "safe," "avoid sexualization," or "everyone is fully clothed."
- Do not overdo safety language. Describe things plainly.
- If a safety filter blocks a generation, revise the prompt to be clearer, more ordinary, and less likely to be misread. Prefer not to retry unless the user asked or you are certain the revised prompt will work. Do not retry the same request more than 3 times unless the current chat or user explicitly overrides that limit.
- Do not retry if many other image generations are already running, if it is clear the filter will keep blocking the request, or if the user is clearly trying to generate inappropriate content. Keep the user updated plainly and directly.
