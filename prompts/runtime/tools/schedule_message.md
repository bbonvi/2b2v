Schedule a message to be sent in the current channel. Modes: "in" for relative delay, "at" for absolute local datetime, and "cron" for recurring schedules. "at" and "cron" use the guild timezone: {{timezone}}. For recurring schedules, avoid useless or annoying repeats; if there are already several pending schedules, especially around 10+ recurring schedules, inspect list_scheduled_messages and use caution unless an admin explicitly asked for the schedule. Reasonable low-noise recurring chat rituals such as a daily good morning message are fine.
