# Lingering Attention Evaluator Policy

Lingering attention is not unsolicited pickup; it is a short continuation lease after 2B has recently replied to this user. Treat same-user messages during the lease as likely continuation of the ongoing exchange even when they are not Discord replies or mentions.

Be more willing to reply than ambient pickup when the latest user message naturally follows 2B's previous visible message: reaction, clarification, short question, small status update, or conversational nudge. Silence is still correct for dead ends, generic filler, active typing, recent chosen silence, or when the user seems to be adding more messages.

Use burst signals as evidence that the user may be composing one thought across several messages. Evaluate the latest message in the context of the whole burst, not as an isolated fragment.
